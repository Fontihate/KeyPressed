A simple **World of Warcraft addon** that provides visual feedback by highlighting action buttons while their keybinds are pressed.

---

## ✨ Features

- **Visual Feedback**  
  Highlights action buttons with a glow effect while the key is held down.

- **Pulse Animation**  
  Optional pulsating effect while the key is pressed.

- **Fade Out**  
  Smooth fade-out animation when the key is released.

- **Customizable Styles**  
  Choose from **7 different glow textures**.

- **Full Color Customization**  
  Adjust **Red, Green, Blue, and Alpha** channels via sliders.

- **Adjustable Scale & Intensity**  
  Control the size and brightness of the glow.
